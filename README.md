# 💫 About Me:

🔭 I'm currently working on Ask Apex — a portfolio app that walks through how agentic AI actually works, scene by scene. Built with React, FastAPI, LangGraph and LlamaIndex. Launching before my internship starts in June.

👯 I'm looking to collaborate on projects that make AI behaviour legible to people who aren't AI engineers. Especially anything in fintech.

🤝 I'm looking for help with LLM evaluation in production — tracing, evals, knowing when your agent is silently wrong.

🌱 I'm currently learning LangGraph, LlamaIndex, vector search with ChromaDB. Incoming Data/AI intern at UBS in June, trying to hit the ground running.

💬 Ask me about getting a React + FastAPI agent demo working without a live backend — I cached the trace and deployed the whole thing statically to Vercel.

⚡ Fun fact: the "under the hood" panel in my portfolio renders a live 3D vector space in Three.js so you can actually see why the AI retrieved that document and not a different one.
